💡 Widget custom.html hỗ trợ datasource tùy chọn. Dữ liệu được inject vào trong iframe:

0. Chọn shape dữ liệu — trong Query config (JSON) thêm key shape với một trong các giá trị single, timeseries, category, table hoặc list để backend lấy đúng dạng dữ liệu.
1. Template biến — dùng {{value}}, {{previous}}, {{shape}}… trực tiếp trong HTML (thay thế tự động từ payload).
2. JavaScript — truy cập window.WIDGET_DATA trong thẻ <script> để xử lý linh hoạt hơn.

Ví dụ: <div>{{value}} / <script>console.log(window.WIDGET_DATA)</script>
